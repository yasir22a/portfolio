A mobile robot designed to be controlled remotely via Bluetooth using a smartphone or other compatible devices. Built with Arduino and HC-05 modules, it can move in multiple directions and perform basic tasks with precise control, offering an interactive platform for learning robotics, understanding wireless communication, and demonstrating practical applications of remote-controlled robotic systems.

Gesture Mouse & Scroll Control

A touch-free interface system that leverages computer vision through OpenCV to detect hand gestures and control cursor movement, scrolling, and basic commands on a computer. By translating natural hand movements into computer interactions, this system allows users to operate devices more intuitively, which is particularly useful for presentations, hygiene-sensitive environments, and for assisting users with limited mobility.

Object Sorting Robot (Arduino-Based System)

An intelligent robot capable of automatically detecting, identifying, and sorting objects based on parameters such as color, size, or shape using sensors and microcontrollers. Pedestrian Crossing System

A smart traffic control system built using ESP32 and IoT technology that detects pedestrians waiting to cross the road through sensors and automatically manages traffic lights to ensure their safe passage. This system reduces unnecessary waiting time for both pedestrians and vehicles, improves overall road safety, and can be integrated into busy city streets, school zones, and other high-traffic areas to create a more organized and efficient traffic flow.
